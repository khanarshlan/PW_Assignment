Differences Between HTML and HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 stands for HyperText Markup Language version 5.
Introduced in the early 1990s. Introduced in 2014 as a major revision of HTML.
Includes fewer elements and attributes compared to HTML5. Includes many new elements, attributes, and APIs, providing more features and flexibility.
Does not support features like video and audio without third-party plugins. Supports native video and audio elements for embedding multimedia content.
